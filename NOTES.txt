
How to Build YetiSDKInstaller in yeti://dev/visual_studio_integration

* Open YetiVSI.sln with VS 2015 Professional

* Install Extension "Wix Toolset Visual Studio 2015 Extension"

* Download file yeti://dev/visual_studio_integration/YetiSDKInstaller/YetiSDK-windows.zip
  > gsutil cp gs://yeti_base_sdk/releases/2019.4devdebian/YetiSDKBase-windows.zip .

  Tip: you can use this to find the zip file:
  > gsutil ls gs://yeti_base_sdk/releases/

* Download file yeti://dev/visual_studio_integration/YetiSDKInstaller/YetiSDK-windows-renderdoc.zip
  > cd %USERPROFILE%\Downloads
  > mkdir tmp 
  > cd tmp
  > gsutil cp -r gs://yeti_builds/sdk_release/sdk/13994.1.42.0/Debug/windows/artifacts/renderdoc/client/* .
  # Then zip "tmp/*", rename to YetiSDK-windows-renderdoc.zip and move it

  Tips: use this URL for browsing:
  https://pantheon.corp.google.com/storage/browser/yeti_builds/sdk_release/sdk/
